# コンテキスト管理ルール

## CLAUDE.md のサイズ
- 200行以内を維持
- 超えたら `.claude/rules/` にトピック別ファイルに分割
- `@docs/filename.md` で詳細を参照（インライン展開するな）

## セッション継続性

### CLAUDE_PROGRESS.md
- **セッション開始:** 必ず最初に読んで前回の状態を復元
- **作業中:** 大きなタスクの各ステップ完了後に更新
- **中断時:** 現在のステータス、残TODO、変更ファイルを記録

### CLAUDE_ISSUE.md
- 非自明なバグ修正は全て記録（症状/原因/解決/予防）
- デバッグ前にこのファイルを確認して既知の問題を除外
- 法則化できるものはパターンとして抽象化

## サブエージェント活用
- Explore: 大規模コードベース検索（メインコンテキスト節約）
- バックグラウンド: テスト実行、ビルドなど長時間タスク
- メイン会話にはサマリーだけ返す

## 圧縮時の保持項目
- 変更したファイル一覧
- 現在のタスク目標
- 重要な判断とその理由
